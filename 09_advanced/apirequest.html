0: UNSENT - The request has not been initialized. - The open() method has not yet been called. 1: OPENED - The request has been set up. - The send() method has not yet been called. 2: HEADERS_RECEIVED - The request has been sent and the headers have been received. - The response status and headers are now available. 3: LOADING - The response is being downloaded. - responseText holds partial data as it is being received. 4: DONE - The request is complete and the response is fully received. - The response can be processed.
